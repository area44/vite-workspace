# Vite Starter

This template provides a minimal setup to get React working in Vite.

## Development

```bash
git clone https://github.com/area44/vite-starter
cd vite-starter
pnpm install
pnpm dev
```
Open your web browser and navigate to [http://localhost:4321](http://localhost:5173)
