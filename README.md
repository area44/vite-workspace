# Vite Workspace

This template provides a minimal setup to get React working in Vite.

## Development

```bash
git clone https://github.com/area44/vite-workspace
cd vite-workspace
pnpm install
pnpm dev
```
Open your web browser and navigate to [http://localhost:5173](http://localhost:5173)
